--- layout: post title: "CTT Tool | Chris Titus Tech" date: 2025-06-26 permalink: /ctt/ --- CTT Tool is an open source project (over 200 contributors) that provides many resources such as the ability to install certain applications, tweak/debloat your system, and much more

What features can be found within CTT Tool?

Program Installation

Provides you with the options to download these applications

  • Browsers (Brave, Chrome, Firefox)
  • Communication (Discord, Zoom, Teams)
  • Development (Git, Docker Desktop, VS Code)
  • Document (Adobe Acrobat Reader, Anki)
  • Games (EA App, Epic Games, Steam)
  • Microsoft tools (Autoruns, One Drive, runtimes)
  • Multimedia tools (Handbreak, iTunes, Spotify)
  • Pro tools (Wireshark, PuTTY, RustDesk)
  • Utilities (7-zip, CPU-Z, HWiNFO)
and much more from all categories.

Debloat/Tweaks

Examples include:

  • Removal of Telemetry/Tracking
  • Disable Recall/CoPilot
  • Network Optimizations
  • Removal of unneeded bloat/applications
  • Aesthetic Changes
and much more

Misc

Allows easy access for:

  • Enabling Windows Features
  • Resetting settings (updates, network, etc)
  • Gives you ability to open legacy Windows panels

Update Adjustments

You have three options provided for update adjustments:

  • Default settings
  • Feature updates delayed by 2 years/security updates installed after 4 days (for systems with group policy only)
  • All updates disabled with an increased security risk

MicroWin Description

On the last page of his tool he places a description for his iso customization tool. Features for it include:

  • Removed telemetry/tracking
  • Fast installation using either a local account or one of your choosing
  • No internet req. for install
  • Apps debloated

Try for yourself

You can gain access to his tool through copy/pasting this command below and running it inside of an elevated(admin) powershell prompt

  • iwr -useb https://christitus.com/win | iex
