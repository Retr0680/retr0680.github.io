Personal compatibility notes for running games via Proton on CachyOS. All entries tested on my system — your mileage may vary.

<!--more-->

| Game | Proton Version | Launch Options | Source | Notes |
| :--- | :--- | :--- | :--- | :--- |
| **Mortal Kombat 1** | **Proton Experimental** | ```PROTON_DLSS_UPGRADE=1 DXVK_NVAPI_DRS_NGX_DLSS_SR_OVERRIDE_RENDER_PRESET_SELECTION=RENDER_PRESET_LATEST bash -c '("$@") & while true; do PID=$(pgrep -u $USER -f "/Mortal Kombat 1/MK12/Binaries/Win64/MK12.exe" | head -n 1); [ -n "$PID" ] && break; sleep 0.1; done; taskset -cp 0-15:2 "$PID"\' -- %command% gamemoderun %command%``` | steam | turn off steam overlay in-game |
| **Tekken 8** | **GE-Proton10-34** | `prime-run %command%` | steam | — |
| **Marvel Rivals** | **proton-cachyos-11.0-20260521-slr-x86_64** | `prime-run %command%` | steam | — |
| **Doom Eternal** | **proton-cachyos-11.0-20260602-slr** | `+com_skipIntroVideo 1` | lutris (pirated) | `ntfs-3g` driver — see config image below |
| **DOOM: The Dark Ages** | **GE-Proton10-26** | `+com_skipIntroVideo 1 WINEDLLOVERRIDES="steam_api64,steamclient64,voices38=n,b" PROTON_DISABLE_LSTEAMCLIENT=1 %COMMAND%` | lutris (pirated) | `ntfs-3g` driver — mount: `sudo mount -t ntfs-3g /dev/sda2 /mnt/Retr0` |
| **MECCHA CHAMELEON** | **proton-experimental** | `WINEDLLOVERRIDES="OnlineFix64=n;SteamOverlay64=n;winmm=n,b;dnet=n;steam_api64=n;winhttp=n,b" %command%` | steam (pirated) | `ntfs-3g` driver — install `Microsoft Visual C++ 2015-2022` via `winetricks` or `protontricks` |
| **LEGO Batman: Legacy** | **Proton-CachyOS Latest** | — | lutris (pirated) | `ntfs-3g` driver — install `Microsoft Visual C++ 2015-2022` via `winetricks` or `protontricks` |
| **Watch Dogs** | **GE-Proton11-1** | — | lutris (pirated) | using `ntfs-3g` driver |
| **Batman: Arkham Origins** | **Proton-CachyOS Latest** | - | lutris(pirated) | using `ntfs-3g` driver |

## Final Decision
by the looks of it i think lutris gives more fps in pirated games compared to steam, but for games like Meccha Chameleon requires steam to play online.
